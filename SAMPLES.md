# Samples / Attribution / Licensing

This file documents the provenance, license terms, and recommended handling for the sampled audio assets used (or referenced) by this project.

IMPORTANT: sample WAVs and other audio assets are third-party content and may have licensing restrictions that differ from the code in this repository. Treat samples separately from source code - check each pack's license before redistributing or publishing the repository publicly.

## Summary (packs mentioned by maintainer)

### Salamander Grand Piano V2
- Source / author: Alexander Holm
- License: Creative Commons Attribution 3.0 (CC BY 3.0)
  - Link: http://creativecommons.org/licenses/by/3.0/
- Technical summary (from pack README):
  - Recorded @ 48 kHz / 24 bit
  - 16 velocity layers, sampled in minor thirds from the lowest A
  - Hammer noise releases and string resonance layers sampled as described in source README
  - Recommended/use notes: originally optimized for LinuxSampler; SFZ tuning values provided in the pack README
- License implications:
  - CC BY 3.0 permits redistribution and reuse (including commercial use), provided proper attribution is given.
  - If you include these WAVs in a public repository, you MUST include clear attribution and a reference to the CC BY 3.0 license.
  - If you modify the samples or convert them to another format, you should continue to provide attribution.
